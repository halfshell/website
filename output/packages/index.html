Package Repository

Halfshell modules are managed by npm using the public registry. A package must be published using the npm keyword "halfshell" to be recognized. They generally fall into one of two categories; commands and themes. Each package should contain at most one theme or command. Grouped command sets can be created as a meta-package with multiple dependencies.
